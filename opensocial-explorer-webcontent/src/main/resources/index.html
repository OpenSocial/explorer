Learn

If you are looking for information on how to use a feature or API from the OpenSocial spec, chances are you can find an example of it in the OpenSocial Explorer. The goal of the OpenSocial Explorer is to enable developers to easily build gadgets, and the various samples in the OpenSocial Explorer are the key to doing that.

Try

Not only can you view the source for all the samples but you can edit the code and re-render the gadget. This allows you to change the code of the samples to gain a deeper understanding of how OpenSocial works.

Build

The samples not only allow developers to learn how to use OpenSocial but give developers a place to get started with their own gadgets. We encourage developers to copy the code out of any of the samples and modify it to be used within their own gadgets.
